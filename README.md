# fb-hack

# Installation 
Install 
```shell
pip install -r requirements.txt
```


For the client, install Angular CLI: 
```bash
sudo npm install -g @Angular/cli
cd client
npm install
ng serve --open
```


Download resources: 
```python
import nltk 
nltk.download('punkt')
nltk.download('averaged_perceptron_tagger')
```
